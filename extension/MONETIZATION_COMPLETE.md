## ðŸ†˜ Troubleshooting

### License Validation Not Working?

**Check:**
1. Product permalink matches in code (`pinterest-scraper`)
2. License keys are enabled in Gumroad
3. Test with real purchase (not fake key)
4. Check console for error messages

### Customers Can't Activate?

**Common issues:**
1. They're typing instead of copy-pasting
2. Extra spaces before/after key
3. Using refunded purchase
4. Extension not updated

**Solution:** Send them support email with steps

### Want to Change Product Permalink?

Update in `license.js` line 7:
```javascript
const PRODUCT_PERMALINK = 'your-new-permalink';
```
